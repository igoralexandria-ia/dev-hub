# DevHub

Uma aplicação web moderna criada para desenvolvedores. O DevHub centraliza comandos essenciais, documentações rápidas e setups completos de diversas tecnologias.

## Objetivo
Construir uma plataforma que funcione como uma ferramenta diária para devs, onde é possível encontrar e copiar rapidamente como iniciar projetos e configurar ambientes (como Next.js, Node.js, Python, etc).

## Funcionalidades Planejadas
- **Dashboard Principal:** Busca rápida, stacks mais populares e comandos mais usados.
- **Sistema de Busca Dinâmico:** Encontre resultados por tecnologia (ex: "next") ou por ação (ex: "instalar").
- **Catálogo de Tecnologias:** Exemplos práticos e descrição dos comandos principais.
- **Stacks Prontas:** Combinações como "Next.js + Tailwind + Prisma" com setup completo.
- **Gerador de Setup (Em Breve):** Assistente interativo para gerar o passo a passo.
- **Favoritos (Em Breve):** Salvar setups e comandos vinculados à sua conta.

## Como Executar o Projeto Localmente

1. Clone o repositório e instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra [http://localhost:3000](http://localhost:3000) no seu navegador.
